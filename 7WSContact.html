Filing an Issue

The Issue Tracker is used for tracking items of work.
If you would like for us to take an action in response to your message, please file the message as an issue in the tracker.

This is explicitly appropriate for reporting data corrections and requests to add new meets, but it is never incorrect to file an issue on the tracker. If you want to send us spreadsheets, please do so through an issue so that your request can be tracked.

For your convenience, you can also file an issue by sending an e-mail to issues@weightliftingstatistics.com

For corrections to data, please include a link to the corrections, preferably from a federation-linked source. Video evidence is also acceptable in the case of federation error or omission.

E-Mail

Please address all issue requests to: issues@weightliftingstatistics.com

If you would just like to talk with us, please address all other inquiries to: updates@weightliftingstatistics.com

If you would like to contribute to the project, please send an email for an invite to the team chat.
Help is always appreciated. Every meet requires manual work, and there's more than enough to go around.
